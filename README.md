## ğŸ¨ Customization

### Colors
The purple color scheme can be customized in `tailwind.config.js`. The theme uses a comprehensive purple palette from 50 to 950.

### Content
Update your information in:
- `src/utils/constants.ts` - Skills, projects, and social links
- `src/components/About.tsx` - Personal description
- `src/components/Hero.tsx` - Hero section content

### Animations
Animations are configured in:
- `tailwind.config.js` - Custom keyframes and animations
- Component files - Framer Motion animations
